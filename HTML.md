# HTML

- ctrl + D   删除光标当前所在行
- ctrl + shift + R    复制当前行到下一行
- ctrl + enter    将光标移动到下一行
- ctrl + shift + enter   将光标定位在上一行
- ctrl + shift + /    注释当前行
- ctrl + R   运行当前网页/刷新当前网页 

```html
<!DOCTYPE html><!-- 文档声明 -->
<html lang="en">
<head>
	<meta charset="UTF-8">
	<title>Document</title>
</head>
<body>
	
</body>
</html>
```

主要包含两部分

- 头部份：主要用来放置一些页面信息
- 体部分：主要来放置html页面内容
